# 🚢 FMC Inspection Dashboard

## 📌 Project Info
This is a Marine Inspection Management Dashboard where clients can submit enquiries and admins can manage inspections and quotations.

---

## ⚙️ Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

---

## 🚀 Run Locally

```sh
# Clone project
git clone <YOUR_GIT_URL>

# Open folder
cd fmc-inspection-dashboard

# Install dependencies
npm install

# Run project
npm run dev
